#Recipe Page (HTML Only)

A simple beginner-friendly HTML project that displays recipe pages using pure HTML.  
This project is designed to practice **semantic HTML structure and content organization** without using CSS or JavaScript.

---

## 📌 Project Overview

This project is a static website built using only HTML.  
It demonstrates how to properly structure multiple recipe pages in a clean and readable format.

Each recipe is written as a separate HTML page and linked together (if multi-page setup is used).

---

##What I Learned

- Basic HTML document structure
- Semantic HTML tags:
  - `<header>`
  - `<main>`
  - `<section>`
  - `<footer>`
- Working with text content
- Creating lists:
  - Ordered lists (`<ol>`)
  - Unordered lists (`<ul>`)
- Adding images using `<img>`
- Using links with `<a>`
- Structuring multiple content sections properly

---

##Features

- Multiple recipe pages (expandable)
- Structured layout using semantic HTML
- Ingredients section
- Step-by-step instructions
- Cooking time and serving details
- External links for references or tutorials
